🎮 GUIDE RAPIDE - MMO 2D JEU DE SURVIE
==========================================

📋 INSTALLATION
---------------
1. Assurez-vous d'avoir Python 3.7+ installé
2. Installez les dépendances : pip install -r requirements.txt
3. Lancez le jeu : python main.py ou python launch.py

🎮 CONTRÔLES DE BASE
-------------------
🚶 DÉPLACEMENT:
  • WASD ou flèches directionnelles : Se déplacer
  • Le joueur ne peut marcher que sur l'herbe (cases vertes)

🛠️ RÉCOLTE DE RESSOURCES:
  • Clic gauche : Récolter arbres, pierres, minerais
  • Portée de récolte : 2 cases autour du joueur
  • Les ressources récoltées vont automatiquement dans l'inventaire

🏗️ CONSTRUCTION:
  • B : Activer/désactiver le mode construction
  • 1 : Sélectionner fondation (coût: 2 bois + 1 pierre)
  • 2 : Sélectionner mur (coût: 1 bois + 2 pierres)
  • Clic gauche (en mode construction) : Placer une structure
  • Portée de construction : 3 cases autour du joueur

📦 INVENTAIRE ET ARTISANAT:
  • I : Ouvrir/fermer l'inventaire
  • TAB (dans l'inventaire) : Changer d'onglet
  • WASD (dans l'inventaire) : Naviguer dans les menus
  • ENTER (dans l'inventaire) : Crafter ou utiliser un objet

🍎 ALIMENTATION:
  • H : Consommer automatiquement de la nourriture
  • La nourriture restaure la santé et réduit la faim

💾 SAUVEGARDE:
  • F5 : Sauvegarder la partie en cours
  • Échap : Retourner au menu principal
  • "Charger Partie" dans le menu : Reprendre la dernière sauvegarde

🌍 ÉLÉMENTS DU MONDE
-------------------
🟢 TERRAINS:
  • Herbe (vert) : Terrain praticable et constructible
  • Arbres (marron) : Source de bois
  • Pierres (gris) : Source de pierre
  • Minerai de fer (gris foncé) : Source de fer
  • Minerai d'or (doré) : Source d'or (rare)
  • Minerai de diamant (bleu clair) : Source de diamant (très rare)
  • Charbon (noir) : Combustible
  • Arbres fruitiers (vert) : Donnent pommes + bois
  • Buissons de baies (violet) : Donnent des baies

🏗️ STRUCTURES:
  • Fondations (jaune) : Base de construction
  • Murs (orange) : Structures défensives

👹 ENNEMIS:
  • Cercles rouges : Ennemis hostiles
  • Détection : 3 cases de rayon
  • 50 PV, 10 dégâts par attaque
  • Attaquent automatiquement le joueur au contact

🎯 OBJECTIFS ET STRATÉGIES
-------------------------
📈 PROGRESSION RECOMMANDÉE:
1. Récolter bois et pierre pour construire une base
2. Chercher du fer pour créer de meilleurs outils
3. Construire des murs pour se protéger des ennemis
4. Explorer pour trouver des ressources rares
5. Développer son artisanat pour créer armes et armures

🛡️ SURVIE:
  • Surveillez votre santé (barre rouge en haut à gauche)
  • Mangez régulièrement pour récupérer des points de vie
  • Évitez les ennemis si vous êtes faible en santé
  • Construisez des murs pour créer des zones sûres

⚒️ ARTISANAT AVANCÉ:
  • Fondez les minerais avec du charbon pour créer des lingots
  • Craftez des armes : épées en bois → fer → or → diamant
  • Craftez des outils : pioches pour miner plus efficacement
  • Créez des armures pour réduire les dégâts

🔧 ASTUCES ET CONSEILS
---------------------
✨ OPTIMISATION:
  • Récoltez toujours du bois et de la pierre en priorité
  • Gardez toujours de la nourriture dans votre inventaire
  • Construisez près des ressources pour réduire les déplacements
  • Sauvegardez régulièrement avec F5

⚠️ ATTENTION:
  • Ne vous éloignez pas trop de votre base sans protection
  • Les ennemis apparaissent au démarrage et restent permanents
  • Certaines ressources (or, diamant) sont très rares
  • La mort fait retourner au menu (pas de respawn automatique)

🎨 PERSONNALISATION:
  • Modifiez config.py pour ajuster la difficulté
  • Changez MAP_WIDTH/MAP_HEIGHT pour des cartes plus grandes
  • Ajustez ENEMY_COUNT pour plus ou moins d'ennemis

📊 PERFORMANCES:
  • Le jeu vise 60 FPS
  • Utilisez test_performance.py pour tester votre système
  • Réduisez la taille de carte si le jeu ralentit

📞 DÉPANNAGE
------------
❌ PROBLÈMES COURANTS:
  • "ImportError" : Installer les dépendances avec pip install -r requirements.txt
  • Jeu lent : Réduire MAP_WIDTH et MAP_HEIGHT dans config.py
  • Écran noir : Vérifier que pygame est correctement installé
  • Erreur de sauvegarde : Vérifier les permissions d'écriture dans le dossier
