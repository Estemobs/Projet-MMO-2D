# Scripts utilitaires

Ce dossier regroupe les scripts de maintenance du projet.

## Lancement
- `launch.py`: verification d'integrite + lancement du jeu (utilise aussi par `../launch.py`).

## Tests manuels
- `test_controls.py`: verifications rapides des controles et de l'initialisation.
- `test_integration.py`: integration GameManager/menu/sauvegarde.
- `test_simulation_complete.py`: simulation complete de sauvegarde/chargement.
- `test_performance.py`: mesure rapide des performances.

## Generation d'assets
- `create_entity_sprites.py`
- `create_large_characters.py`
- `create_missing_sprites.py`
- `create_tombstone.py`
- `generate_characters_and_items.py`
- `generate_missing_items.py`
- `generate_natural_sprites.py`
- `generate_sprites.py`

Remarque: ces scripts ne sont pas necessaires pour simplement jouer.
